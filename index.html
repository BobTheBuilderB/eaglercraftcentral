Mods & Resource Packs

EaglerForge Mods

EaglerForge allows some client-side mods to be used. These are typically packaged as `.js` files. Finding mods can be tricky and often requires searching through communities on Discord, GitHub, or specialized websites.

How to find them: A great resource for mods is Eaglerrinth. You can also search on GitHub or YouTube. Be cautious about where you download files from.

Resource Packs

You can use standard Minecraft 1.8.8 or 1.12.2 resource packs! Eaglercraft has a built-in feature to load them. Just download a compatible pack, and in the Eaglercraft settings, you can select and load the `.zip` file from your computer.

How to find them: Search for "Minecraft 1.8.8 resource packs" (or 1.12.2) on websites like Planet Minecraft, CurseForge, and Modrinth.
